# Glossaire Windows Server

Ce glossaire couvre les termes essentiels de la formation Windows Server Mastery.

---

## A

**ACL (Access Control List)**
: Liste définissant les permissions d'accès à un objet (fichier, dossier, registre).

**Active Directory (AD)**
: Service d'annuaire Microsoft pour la gestion centralisée des identités et ressources.

**AD DS (Active Directory Domain Services)**
: Rôle serveur qui implémente Active Directory.

**ADFS (Active Directory Federation Services)**
: Service d'authentification fédérée pour le SSO entre organisations.

**Azure AD Connect**
: Outil de synchronisation entre AD on-premises et Azure AD.

---

## B

**BitLocker**
: Technologie de chiffrement de disque intégrée à Windows.

**BranchCache**
: Technologie de mise en cache pour optimiser la bande passante WAN.

---

## C

**CAL (Client Access License)**
: Licence requise pour chaque utilisateur ou appareil accédant à un serveur Windows.

**Cluster**
: Groupe de serveurs travaillant ensemble pour la haute disponibilité.

**CSV (Cluster Shared Volume)**
: Volume de stockage partagé entre les nœuds d'un cluster.

---

## D

**DC (Domain Controller)**
: Serveur hébergeant la base de données Active Directory.

**DFSR (DFS Replication)**
: Service de réplication de fichiers entre serveurs.

**DHCP (Dynamic Host Configuration Protocol)**
: Service d'attribution automatique d'adresses IP.

**DNS (Domain Name System)**
: Service de résolution de noms en adresses IP.

**DSC (Desired State Configuration)**
: Framework PowerShell pour la gestion de configuration déclarative.

---

## F

**Failover Clustering**
: Technologie de haute disponibilité avec basculement automatique.

**Forest**
: Niveau le plus élevé dans la hiérarchie AD, contenant un ou plusieurs domaines.

**FSMO (Flexible Single Master Operations)**
: Rôles spéciaux dans AD nécessitant un seul maître.

---

## G

**GPO (Group Policy Object)**
: Objet contenant des paramètres de configuration appliqués aux utilisateurs/ordinateurs.

**GPMC (Group Policy Management Console)**
: Console de gestion des stratégies de groupe.

**GUI (Graphical User Interface)**
: Interface graphique utilisateur.

---

## H

**Hardening**
: Processus de sécurisation d'un système en réduisant sa surface d'attaque.

**Hyper-V**
: Hyperviseur Microsoft pour la virtualisation.

---

## I

**IIS (Internet Information Services)**
: Serveur web Microsoft.

**iSCSI**
: Protocole de stockage réseau sur IP.

---

## K

**Kerberos**
: Protocole d'authentification par défaut dans les environnements AD.

---

## L

**LAPS (Local Administrator Password Solution)**
: Solution de gestion des mots de passe admin locaux.

**LDAP (Lightweight Directory Access Protocol)**
: Protocole d'accès aux annuaires.

**LCM (Local Configuration Manager)**
: Moteur DSC sur chaque nœud géré.

---

## N

**NLB (Network Load Balancing)**
: Service de répartition de charge réseau.

**NPS (Network Policy Server)**
: Implémentation Microsoft de RADIUS.

**NTFS**
: Système de fichiers Windows avec support des permissions avancées.

---

## O

**OU (Organizational Unit)**
: Conteneur dans AD pour organiser les objets et appliquer des GPO.

---

## P

**PDC Emulator**
: Rôle FSMO gérant la synchronisation du temps et les changements de mot de passe.

**PKI (Public Key Infrastructure)**
: Infrastructure de gestion des certificats numériques.

**PowerShell**
: Shell et langage de scripting orienté objet de Microsoft.

---

## Q

**Quorum**
: Mécanisme de vote dans un cluster pour éviter le split-brain.

---

## R

**RADIUS**
: Protocole d'authentification réseau (Remote Authentication Dial-In User Service).

**RBAC (Role-Based Access Control)**
: Contrôle d'accès basé sur les rôles.

**RDP (Remote Desktop Protocol)**
: Protocole de bureau à distance Microsoft.

**ReFS (Resilient File System)**
: Système de fichiers moderne conçu pour la résilience des données.

**RODC (Read-Only Domain Controller)**
: Contrôleur de domaine en lecture seule pour les sites distants.

**RSAT (Remote Server Administration Tools)**
: Outils d'administration à distance pour Windows Server.

---

## S

**S2D (Storage Spaces Direct)**
: Solution de stockage hyperconvergé Microsoft.

**SAM (Security Account Manager)**
: Base de données des comptes locaux Windows.

**SID (Security Identifier)**
: Identifiant unique pour chaque objet de sécurité Windows.

**SMB (Server Message Block)**
: Protocole de partage de fichiers Windows.

**SSO (Single Sign-On)**
: Authentification unique pour accéder à plusieurs services.

**SYSVOL**
: Dossier partagé contenant les GPO et scripts de connexion.

---

## T

**Tiering Model**
: Modèle de sécurité séparant les comptes privilégiés par niveau.

---

## U

**UPN (User Principal Name)**
: Identifiant utilisateur au format email (user@domain.com).

---

## V

**VHD/VHDX**
: Format de disque virtuel Microsoft.

**VLAN (Virtual LAN)**
: Réseau local virtuel pour segmenter le trafic.

**VPN (Virtual Private Network)**
: Réseau privé virtuel sécurisé.

---

## W

**WAC (Windows Admin Center)**
: Interface web moderne d'administration Windows Server.

**WDS (Windows Deployment Services)**
: Service de déploiement d'images Windows.

**WinRM (Windows Remote Management)**
: Service de gestion à distance basé sur WS-Management.

**WSUS (Windows Server Update Services)**
: Service de gestion des mises à jour Windows.

---

## X

**X.509**
: Standard pour les certificats numériques.
